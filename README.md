# BeatBond 

A full-stack social music platform.

## Live Demo
https://subtle-mousse-61ef38.netlify.app/

## Features
- User authentication
- Create playlists
- Discover public playlists
- Follow users

## ech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## How to Run Locally
1. Clone the repo
2. Install dependencies:
   npm install
3. Create .env file (see .env.example)
4. Run server:
   npm start
