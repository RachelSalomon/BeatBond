# BeatBond

BeatBond is a full-stack social music web application that allows users to create playlists, manage songs, and interact with other users through a simple social system.

## Features

- User registration and login
- Create playlists
- Add and remove songs
- Delete playlists
- Discover public playlists
- Follow users
- View followers and following

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express
- MongoDB

## How to Run

1. Go to server folder:
   cd server

2. Install dependencies:
   npm install

3. Create a .env file:
   MONGO_URI=your_mongo_connection
   JWT_SECRET=your_secret
   PORT=5000

4. Run server:
   node server.js

5. Open client:
   open client/index.html
