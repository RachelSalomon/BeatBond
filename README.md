# BeatBond 🎵

A full-stack social music platform.

##  Live Demo
https://beatbond.netlify.app

##  Features
- User authentication
- Create playlists
- Discover public playlists
- Follow users

##  Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

##  How to Run Locally
1. Clone the repository:
   git clone https://github.com/RachelSalomon/BeatBond.git

2. Install dependencies:
   npm install

3. Create a `.env` file (see `.env.example`)

4. Run the server:
   npm start
